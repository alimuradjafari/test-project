# Project

OneText (onetext.com) is a conversational commerce platform designed to increase revenue by **allowing** customers to purchase directly through SMS, bypassing traditional checkout carts. It features AI-powered replies, automated campaigns, and secure payment integration with tools like Stripe and PayPal to streamline the shopping experience.
